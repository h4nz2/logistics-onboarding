# Welcome Step

## Purpose
Introduce the user to the onboarding process and set expectations for what comes next.

## Guidance
- Greet the user and summarize the onboarding flow: they will configure restocking parameters, add their products and suppliers, and optionally connect integrations.
- Mention that some steps are mandatory (lead time, stock days, forecasting) and others are optional but recommended.
- If the company already has products or vendors in the system (e.g. from an integration sync), acknowledge that and explain which steps may already be partially done.
- Estimate the onboarding will take 10-15 minutes for basic setup, longer if they have files to upload.

## What to tell the user
- This step is just informational — they only need to click "Complete" to move on.
- Briefly preview what the next steps will ask for, so the user can gather any needed information (e.g. supplier details, PO spreadsheets).
